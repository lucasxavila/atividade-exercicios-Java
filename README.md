# Atividade - Programação de Soluções Computacionais

Atividade Avaliativa 02 - 50 exercícios, para praticar Java.

## Pré-requisitos

Certifique-se de ter o [Java Development Kit (JDK)](https://www.oracle.com/java/technologies/downloads/) instalado na sua máquina. 

Para verificar se o Java está instalado, rode no terminal:

```bash
java -version
```

### Como executar as atividades:

Clone o repositório:
```bash
https://github.com/lucasxavila/atividade-exercicios-Java.git
```
Entre na pasta do projeto:
```bash
cd atividade-exercicios-Java
```
Crie a pasta de destino (se for a primeira vez):
```bash
mkdir bin
```
Compile o arquivo Java:
```bash
javac -d bin src/Atvd1.java
```
Execute o programa:
```bash
java -cp bin Atvd1
```

- Nota: Para testar os outros exercícios, basta substituir Atvd1 pelo nome do arquivo correspondente à atividade desejada (ex: Atvd2, Atvd10, etc).
